算术运算符

+	加	两个对象相加

*	乘	两个数相乘或是返回一个被重复若干次的字符串
/	除	x 除以 y
%	取模	返回除法的余数
**	幂	返回 x 的 y 次幂

算术运算符
==	等于：比较对象是否相等
!=	不等于：比较两个对象是否不相等
>	大于：返回 x 是否大于 y
<	小于：返回 x 是否小于 y
>=	大于等于：返回 x 是否大于等于 y
<=	小于等于：返回 x 是否小于等于 y
算术运算符
=	赋值运算符	c = a + b ：将 a + b 的运算结果赋值为 c
+=	加法赋值运算符	c += a ：等效 = c + a
-=	减法赋值运算符	c -= a ：等效于 c = c - a
*=	乘法赋值运算符	c *= a ：等效于 c = c * a
/=	除法赋值运算符	c /= a ：等效于 c = c / a
%=	取模赋值运算符	c %= a ：等效于 c = c % a
**=	幂赋值运算符	c **= a ：等效于 c = c ** a
//=	取整除赋值运算符	c //= a ：等效于 c = c // a
